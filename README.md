## How to use the app

**Step #1** - Clone the project

```bash
$ git clone https://github.com/sandramsc/storefront.git
```

**Step #2**

```
cd client
```

- Install dependencies: `npm i` OR `yarn`

- To start the webapp, run: `npm start` OR `yarn start`
